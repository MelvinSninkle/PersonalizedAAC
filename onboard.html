One more tip: lock the tablet into My World

For younger kids we recommend setting up the child's tablet so the board is always ready and can't be left — the screen stays on, the child can't wander into other apps, and the tablet stays a talker, not a toy.

iPad — Guided Access:

  1. Settings → Accessibility → Guided Access → turn on, set a passcode.
  2. Open My World, then triple-click the side (or home) button and tap Start.
  3. Settings → Display & Brightness → Auto-Lock → Never so the board stays awake.

Android tablet — App pinning:

  1. Settings → Security & privacy → App pinning (sometimes under "More security settings") → turn on.
  2. Open My World, open Recents, tap the app's icon → Pin.
  3. Settings → Display → Screen timeout → the longest setting.
